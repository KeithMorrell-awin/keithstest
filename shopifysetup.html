1. Step One: Getting the app

  • 1. Go to the Shopify store and click Add app on the Awin page
  • 2. Click Install app
  • 3. Enter the Advertiser ID provided to you by the Integrations team.
  • You can find this by logging onto the Awin UI and clicking on Account > Overview, where the Advertiser ID will be shown underneath your account name. (If you encounter any issues when entering their Advertiser ID into the plugin, please try in a private/incognito window.)
  • 4. Click Save.

If you’re not installing the plugin, please visit the Awin Technical Integration Wiki. If you’re using deduplication or tag manager, please use the Developer Guide Wiki for more options.
